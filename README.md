Este é um quiz interativo em Pygame que te ajuda a descobrir qual é o Pokémon inicial perfeito para você! As opções são Charmander, Squirtle ou Bulbasaur.

## Como Jogar

1.  **Inicie o Jogo:** Execute o arquivo principal do jogo. Você verá um botão de "Start" para começar.
2.  **Responda às Perguntas:** Clique no botão de "Start" e responda às perguntas de múltipla escolha, clicando nas opções A, B ou C.
3.  **Descubra seu Pokémon:** Ao final do quiz, o jogo revelará qual Pokémon inicial foi o escolhido para você!

## Requisitos

*   Python 3.x
*   Pygame

## Como Executar

1.  Clone este repositório:
    ```bash
    git clone [https://github.com/everyvini/pokemon_inicial.git](https://github.com/everyvini/pokemon_inicial.git)
    ```
2.  Instale o Pygame:
    ```bash
    pip install pygame
    ```
3.  Execute o jogo:
    ```bash
    python pokemon_version1.py
    ```
